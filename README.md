# Sistema de Boletim Escolar em Python

Projeto desenvolvido em Python para gerenciamento de boletim escolar, permitindo o cadastro de estudantes, lançamento de notas e faltas por disciplina, cálculo automático de médias, frequência e determinação do status final do aluno.

## Funcionalidades

* Cadastro de estudantes
* Registro de notas por bimestre
* Controle de faltas
* Cálculo de média bimestral e soma anual
* Verificação de frequência
* Determinação automática do status:

  * Aprovado
  * Recuperação
  * Reprovado por Nota
  * Reprovado por Faltas
* Geração de boletim final detalhado

## Tecnologias Utilizadas
